# Image Tagging Project

## Description
This project classifies images into:
- Cars
- Cats
- Dogs

## Technologies Used
- Python
- TensorFlow
- NumPy
- Gradio

## Model
CNN (Convolutional Neural Network)

## How to Run
1. Upload an image.
2. Click Submit.
3. The model predicts whether the image is a car, cat, or dog.
